# Портфолио разработчика · Мороз Роман

Это моё персональное веб-портфолио, созданное для демонстрации моих навыков и проектов в области программирования и разработки игр. Сайт представляет собой одностраничное приложение с адаптивным дизайном и поддержкой двух тем: «Космос» и «Солнечный».

## 🌟 Особенности

*   **Две темы оформления:** Переключайтесь между тёмным космическим и светлым солнечным режимами для комфортного просмотра в любое время суток.
*   **Адаптивный дизайн:** Сайт корректно отображается на любых устройствах: от смартфонов до настольных мониторов.
*   **Интерактивные элементы:** Карточки навыков и проектов анимированы и реагируют на наведение курсора.
*   **Атмосферный фон:** В космической теме присутствует мерцающее звёздное небо и плавно парящая луна, а в солнечной — тёплое сияющее солнце.
*   **Плавная анимация:** Все переходы между темами и состояниями элементов сопровождаются красивой анимацией.

## 🛠️ Технологии

*   **HTML5:** Семантическая разметка страницы.
*   **CSS3:** Современные возможности CSS, включая кастомные свойства (CSS-переменные), Flexbox, Grid, фильтры, анимации и медиа-запросы для адаптивности.
*   **JavaScript (ES6):** Управление переключением тем, создание анимированного фона с частицами (canvas), плавная навигация.
